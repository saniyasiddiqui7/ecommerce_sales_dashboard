# Amazon Sales Dashboard 📊

An interactive Power BI dashboard that analyzes Amazon sales data to uncover insights into revenue, profit, product performance, and regional trends.

## 🚀 Project Overview

This project focuses on transforming raw sales data into an interactive dashboard using Power BI. It enables users to monitor key business metrics and identify trends that support data-driven decision-making.

The dashboard answers important business questions such as:

- What is the total sales revenue and profit?
- Which product categories and sub-categories perform best?
- Which products generate the highest sales?
- Which regions contribute the most revenue?
- How do sales and profit change over time?

## 🛠️ Tools & Technologies Used

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- CSV Dataset

## 📌 Key KPIs

- Total Sales
- Total Profit
- Total Orders
- Average Order Value
- Profit Margin %

## 📈 Dashboard Insights

- Identified top-performing categories and products.
- Analyzed monthly sales and profit trends.
- Compared regional sales performance.
- Highlighted the most profitable products.
- Evaluated overall business profitability.

## 📊 Dashboard Features

- Interactive KPI Cards
- Sales and Profit Trend Analysis
- Category and Product Performance Charts
- Regional Analysis
- Dynamic Slicers and Filters

## ▶️ How to Use

1. Clone this repository:
   ```bash
   git clone https://github.com/saniyasiddiqui7/ecommerce_sales_dashboard.git
   ```

2. Open `Sales_performance_dashboard.pbix` in Power BI Desktop.

3. If prompted, reconnect the dataset to `orders.csv`.

4. Refresh the data and explore the dashboard.
